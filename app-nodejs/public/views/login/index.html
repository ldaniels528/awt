Welcome to the AWT

The Azure Workload Tracker (AWT) portal is an interface for the Microsoft Solutions professional (CSA, DSA, TSP, PSS) and managers to track Azure workloads and more importantly the consumption generated via those workloads in a single easy-to-use application.

The goals of this project are to provide:
  • A comprehensive way to track Azure workloads and consumption.
  • A means to associate internal consumption reports (e.g. the Azure Health report) to workloads that are still "in-flight".
  • A shared portal to share ideas between CSA/DSA/TSP's and others.
